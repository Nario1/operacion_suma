## 🧠 ¿Qué hace este script?

<img align="right" src="https://i.imgur.com/UFTn35v.png" width="250" style="margin-left: 20px; border-radius: 10px;" />

Este pequeño script en Python realiza lo siguiente:

- Solicita al usuario **dos números**.
- Valida que ambos sean valores válidos (`int` o `float`).
- Realiza la **suma** y muestra el resultado.
- En caso de error, muestra un mensaje claro y vuelve a solicitar la entrada.

✅ Sin dependencias externas  
✅ Ideal para principiantes  
✅ Totalmente interactivo

# Aplicación operación Suma
## Descripción
El script implementa la suma de dos números y realiza la validación de los operandos
## Funcionalidad
- Suma dos operandos
- **Verifica** que los operandos sean números (int o float)
